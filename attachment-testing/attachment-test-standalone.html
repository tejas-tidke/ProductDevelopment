Standalone Attachment Testing Page

Test the attachment functionality with local storage and Jira integration

Testing Instructions

  1. Select a test file (any type/size)
  2. Choose test type:
    • "Create Issue + Attach" - Creates a new procurement request and attaches the file
    • "Attach to Existing Issue" - Attaches the file to an existing issue
  3. Click the test button to execute
  4. Check the results panel for success/failure messages
  5. Verify in Jira that the attachment was properly uploaded

API Configuration

This test page communicates with the backend API at: http://localhost:8080
